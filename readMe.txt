Clev3r เวอร์ชั่นนี้ดัดแปลงมาจากเวอร์ชั่น 1.7.1.0 โดยได้เพิ่ม/แก้ไข

- PongTools (lib) มีฟังก์ชั่นพื้นฐานสำหรับควบคุมหุ่นยนต์สำหรับการแข่ง WRO
- PongTools (Help) เป็นหน้าต่างอธิบายคำสั่งต่างๆว่าใช้งานอย่างไร
- แทนที่ภาษาEnglishด้วยภาษาไทย เนื่องจาก report Error บางอย่างไม่สามารถModภาษาได้ (ในโปรแกรมจะเห็นตัวเลือกภาษาเป็น ไทย)
- ปรับแต่งธีมสีเล็กน้อย (เลือกได้ว่าจะลงไหม) # หากลงแล้วต้องการเปลี่ยนทีหลัง ให้คลิกขวาที่ "หน้าหลัก" --> "การตั้งค่า"

โดยสามารถศึกษาการใช้งานเบื้องต้นได้ใน "Programs\Program.bp" 
**หากจะเขียนโปรแกรมในโฟลเดอร์ที่อยู่ Path อื่นให้ทำการคัดลอกโฟลเดอร์ Modsไปด้วยเพื่อใช้งาน PongTools
